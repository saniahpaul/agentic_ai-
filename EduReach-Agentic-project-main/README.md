# 🎓 Mysore College — AI-Powered Admissions Platform

An intelligent college admissions platform that helps students find the right engineering branch using AI.

🔗 **Live Demo**: [https://edu-reach-agentic-project.vercel.app](https://edu-reach-agentic-project.vercel.app)

---

## ✨ What it does

- 🤖 **AI Chatbot** — Answers questions about courses, fees and placements. Recommends best branch based on KCET rank instantly
- 📞 **AI Voice Counselor** — Calls students, collects their KCET rank and 12th marks, gives personalized branch recommendations
- 📧 **Auto Email** — After every call, Gemini AI reads the transcript and sends a personalized email with branch recommendation and fee details
- 📄 **Brochure Download** — Downloadable college brochure with all program details

---

## 🛠️ Tech Stack

**Frontend** — React, TypeScript, Tailwind CSS

**Backend** — Node.js, Express, MongoDB

**AI** — LangChain, Google Gemini, MongoDB Atlas Vector Search, Vapi

**Deployed on** — Vercel + Render

---

## 🚀 Setup

```bash
# Clone

# Install & run backend
cd edureach-platform/server
npm install && npm run dev

# Install & run frontend
cd edureach-platform/client
npm install && npm run dev
```

Add `.env` files with your API keys for MongoDB, Google AI, Vapi and Gmail.
